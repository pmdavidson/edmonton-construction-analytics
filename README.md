# Edmonton Construction & Housing Activity Intelligence

## Overview

This project analyzes City of Edmonton building permit activity using Python, SQL, SQLite, and dashboard-ready analytics workflows.

The goal is to identify:
- housing growth trends
- construction activity patterns
- neighbourhood development concentration
- permit seasonality
- construction value trends

## Tools Used

- Python
- pandas
- SQLite
- SQL
- matplotlib
- Jupyter Notebook

## Project Pipeline

1. Extract permit data from Edmonton Open Data
2. Clean and standardize permit records
3. Load data into SQLite
4. Run SQL analytics queries
5. Export dashboard-ready summary tables
6. Generate exploratory analysis visualizations

## Key Questions

- Which neighbourhoods are growing fastest?
- How has permit activity changed over time?
- Which permit categories dominate development activity?
- Are there seasonal construction patterns?
- Where is housing growth concentrated?
