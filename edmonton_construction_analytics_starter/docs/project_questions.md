# Project Questions

## Main question

Where is Edmonton's construction and housing activity growing, and what types of development are driving it?

## Subquestions

1. How has total permit activity changed over time?
2. Which neighbourhoods have the most permit activity?
3. Which neighbourhoods are adding the most units?
4. Which permit types dominate the dataset?
5. Are permit volumes seasonal?
6. Which neighbourhoods have accelerated recently?
7. Are there zoning or permit-type patterns behind growth?
8. What limitations exist in using issued permits as a proxy for construction activity?
